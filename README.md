# Tamheer ICT Training

This repository documents the hands-on ICT experience gained during my Tamheer training program.

## Overview
- Role: ICT Trainee
- Duration: 6 months
- Environment: Enterprise IT & Networking
- Scope: Support, Systems, and Networking

## Responsibilities
- Handling IT support tickets received from the Service Desk for employees across multiple companies.
- Installing and troubleshooting software and operating system issues.
- Resolving user login, VPN, and connectivity issues.
- Formatting and preparing devices for new employees.
- Creating and managing users and email accounts using Active Directory.
- Supporting extension and internal communication systems for multiple sites.

## Network Experience
- Participated in supervised network training sessions.
- Performed hands-on network labs using Cisco Packet Tracer, including:
  - VLAN configuration
  - Switching fundamentals
  - OSPF routing
  - Basic BGP routing
- Reviewed real-world network configurations for enterprise environments.
- Identified design and configuration issues and discussed improvements with the network supervisor.
- Assisted in troubleshooting internet connectivity issues.

## Tools & Technologies
- Cisco Packet Tracer
- Cisco IOS
- Active Directory
- VPN tools
- Windows systems
- Networking fundamentals (Switching & Routing)

## Key Learnings
- Practical understanding of enterprise IT support workflows.
- Improved troubleshooting and analytical skills.
- Exposure to real network design and configuration practices.
- Understanding the importance of secure and scalable network design.
